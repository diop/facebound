# Facebound
Face detection web app

## Getting Started
```
git clone https://github.com/diop/facebound.git
cd facebound
```

### Setup Python Env
```
pip install virtualenv
virtualenv venv
source venv/bin/activate
```
### Install Packages
```
pip install -r requirements.txt
```

### Run the app
```
streamlit run app.py
```
